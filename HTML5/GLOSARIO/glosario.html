Glosario de términos vistos en las últimas sesiones de clase

  • Etiqueta <html>: Contenedor principal de una página web donde se define toda su estructura.
  • Etiqueta <footer>: Elemento HTML que define el pie de página de un sitio web.
  • Encabezados (<h1>, <h2>, etc.): Etiquetas HTML que estructuran el contenido mediante títulos de distintos tamaños.
  • Visual Studio Code: Editor de código desarrollado por Microsoft, ampliamente usado para programar en HTML y otros lenguajes.
  • Estructura de archivos: Organización de carpetas y archivos dentro de un proyecto web para facilitar su desarrollo y mantenimiento.
  • Metaetiqueta (<meta>): Etiqueta HTML utilizada para definir metadatos, como palabras clave o descripciones de la web.
  • Formulario (<form>): Elemento HTML que captura datos ingresados por el usuario mediante campos de entrada.
  • Listas (<ul>, <ol>, <li>): Etiquetas HTML que organizan elementos en listas ordenadas o desordenadas.
  • Imagen (<img>): Etiqueta HTML utilizada para mostrar imágenes dentro de una página.
  • Enlace (Link): Elemento HTML que permite navegar entre diferentes páginas o sitios web.
  • Atributo: Propiedades dentro de una etiqueta HTML que añaden información adicional o modifican su comportamiento.
  • Etiqueta (Tag): Elementos HTML, representados entre <>, que organizan el contenido dentro de una web.
  • CSS (Cascading Style Sheets): Lenguaje de estilos que permite definir la apariencia y el diseño de una página web, controlando colores, tipografía, márgenes, disposición de elementos y más.
  • JavaScript: Lenguaje de programación que añade interactividad y dinamismo a las páginas web.
  • ID (id): Atributo en HTML que identifica de manera única a un elemento, permitiendo aplicarle estilos específicos con CSS.
